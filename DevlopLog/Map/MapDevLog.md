# 맵
## 계획
- 맵 방식
    - 타일 맵 방식 채용
    - 맵 클릭시 해당 타일만 하이라이트 하는 방법을 찾아야 함함
- 방어 포인트
    - 적군이 들어갈 경우 포인트는 감소
    - 방어 포인트가 0이 될 경우 전투 패배
- 적 등장 포인트
    - 적군이 시간에 따라 소환됨
- 배치 불가 타일
  - 장식품 : 말 그대로 장식
  - 적 동선선
- 배치 가능 타일
  - 언덕 배치 : 터렛 배치 가능
## 제작 방식
- 타일맵을 3개의 구조로 분할
    - 지상 타일
    - 공중 타일
    - 장식 타일
